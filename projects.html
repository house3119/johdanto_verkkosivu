Projektit

1. List Companion

List Companion on web-sovellus, jonka rakensin kurssin CS50's Web Programming with Python and JavaScript lopputyöksi. List Companion on mobile-first -suunnitteluperiaatteen mukaan rakennettu muistilistasovellus, jonka avulla on helppoa jakaa muistilistoja esim. ystävien kanssa. Backend tehty Djangolla ja tietokantana toimii SQLite3.

Kurssin CS50: Introduction to Computer Science lopputyö. Project Minesweeper on allekirjoittaneen versio klassisesta miinaharavasta web-sovelluksena. Pelin logiikka rakennettu Javascriptilla ja backend Flaskilla. Tietokantana SQLite3.
